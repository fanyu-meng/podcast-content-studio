# 嘉宾资料

"爸，咱俩在干什么，也许没有成功之前，就只有咱俩自己知道咱们在干什么。"

**核心标签：** 指弹吉他演奏家 / 2004年生的欧洲吉他奖冠军 / 13岁退学的甘肃白银少年 / 德累斯顿音乐学院在读

**嘉宾亮点：**

* **一夜成名：** 2017年，13岁的刘嘉卓翻弹AC/DC《Thunderstruck》的视频浏览量过亿，高晓松转发留言"看完，砸琴！下半辈子只吹口哨"。视频其实是学校联欢会上没人听他弹琴，他"一怒之下"从1楼弹到4楼，被老师录下来传到网上的。
* **全家All In：** 父亲刘晓龙曾是贝斯手，月薪100多块时花1100元给儿子拉网线学指弹，后来辞职做全职爸爸，全家靠妻子2000元月薪生活。嘉卓走红后举家从甘肃白银北漂到北京。
* **从中国冠军到欧洲冠军：** 2017年获WAGF（中国最大原声吉他赛）冠军，史上最年轻；2024年获第六届欧洲吉他奖冠军，首位中国人、最年轻获奖者。从"开个琴行当吉他老师"的梦想到站上欧洲领奖台。
* **现在进行时：** 在德国德累斯顿音乐学院深造，2024年获德国国家奖学金。第三张专辑《Soliloquy》由德国厂牌Doctor Heart Music出版。音乐风格融合爵士、放克、流行、世界音乐，Slash和Andy McKee都给过高度评价。

# 采访大纲

**一、 暖场 + 访谈**

1. 自我介绍
2. 高晓松当年看完你的视频说"看完，砸琴！下半辈子只吹口哨"——13岁的你看到这条评论的时候，第一反应是什么？你当时知道高晓松是谁吗？
3. 听说那个爆红的视频其实是你在学校联欢会上弹琴没人听，"一怒之下"从1楼弹到4楼被老师录下来的——所以你当时是在对着空气演奏？
4. 你以前逮着谁都问"你想听我弹琴吗"，连来收水电费的阿姨都不放过。现在21岁了，还会这样吗？

**二、 13岁退学：那个决定**

1. 初一读完就退学练琴——这个决定是你自己提出来的，还是爸爸的主意？家里当时有没有吵过架？
2. 你说过"我觉得天赋至多有一点，不是很多的，我只是别人在玩的时候，我更多的在练琴"。但你13岁就退学了——一个觉得自己"天赋不多"的人，哪来的底气退学？
3. 你爸当时月薪100多块，花了1100——十个多月的工资——给你拉了根网线学指弹。后来又辞职做全职爸爸，全家靠你妈2000块过日子。你小时候知道家里有多紧吗？还是长大以后才意识到的？
4. 有篇报道里你爸说"如果儿子因为生活压力放弃梦想，他可能以后就真的失去这个机会了"——但换个角度想，如果你没成功呢？你和你爸有没有聊过这个"万一"？
5. 有人被质疑你的视频是后期加速，你把闹钟摆在屏幕前重新录了一遍来证明自己。13岁的你被质疑的时候，是愤怒多一点还是委屈多一点？
6. 退学之后第一年最难的是什么？是练琴本身，还是看着同龄人都在上学？

**三、 从白银到德累斯顿：一个人走出去**

1. 从甘肃白银到北京，再到德国德累斯顿——每一次搬家，你失去了什么，又得到了什么？
2. 你之前说过想考德累斯顿音乐学院——是什么时候产生的想法？一个从甘肃小城出来的、没读完初中的孩子，是怎么知道德累斯顿这个地方的？
3. 到了德国之后，你是第一次长期离开父母吗？你爸陪了你这么多年，突然不在身边是什么感觉？
4. 你的吉他是从网上看视频自学的，50多首曲子都是"反复慢放回放"学会的，到了德累斯顿才开始系统学习——有没有哪个瞬间觉得"原来之前学的方式全是错的"？
5. 2024年拿到欧洲吉他奖冠军的那天晚上，你第一个电话打给了谁？

**四、 21岁的刘嘉卓**

1. 你的三张专辑分别叫《新的一天》、《17》、《Soliloquy》（独白）——从"新的一天"到"独白"，这个变化是刻意的吗？21岁的你想通过音乐说什么？
2. 指弹吉他在中国的受众还是挺小众的。你的视频过亿，但真正听指弹的人可能只是其中很小一部分——你介意这种"看热闹 vs 真的懂"的落差吗？
3. 如果现在有个13岁的孩子跑来跟你说"我也想退学弹吉他"，你会怎么跟他说？
4. 你现在一个人在德国，除了音乐，生活里还有什么是你在乎的？
