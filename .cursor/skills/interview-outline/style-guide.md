# 采访大纲写作风格指南

## 核心原则

**具体 > 笼统，好奇 > 严肃，追问 > 罗列**

采访大纲的目标是让对话自然展开，让嘉宾讲出故事，不是列一个 FAQ。

---

## 嘉宾资料写作

### 规则

1. **核心标签**用 `/` 分隔，3-5 个关键词，突出身份反差或跨界
2. **嘉宾亮点**用 bullet points，每条带 bold 子标签
3. 每条亮点必须包含**具体数字或细节**，不说"丰富经验"
4. 亮点描述后要有 "so what" —— 为什么这对采访有价值
5. 如果能找到嘉宾的原话/金句，放在资料部分最前面

### 示例对比

**坏（简历体）：**
> **核心标签：** 资深管理者 / 创业者 / 职场导师
>
> * **管理经验：** 拥有多年大厂管理经验，带领过大型团队
> * **创业经历：** 创办了一家职场教育公司

**好（故事体）：**
> **核心标签：** 前亚马逊 L8 总监 / 硅谷高管教练 / 职场"非职权影响力"专家
>
> * **硬核背景：** 密歇根大学商学院/数学/经济多面手，从顶级 IC 一路杀到亚马逊 L8 总监，管理过 1000+ 人的全球团队。
> * **痛点洞察：** 深谙海外华人职场"天花板"的真相。他认为华人职场的瓶颈往往不在于语言，而在于**对权力游戏规则的误读**以及**沟通肌肉记忆的缺失**。

## 采访问题设计

### 五条铁律

1. **引用具体细节**：每个问题必须包含至少一个只属于这位嘉宾的具体信息（数字、地名、引用、事件）。如果把嘉宾名字遮住，这个问题不应该能问任何其他人。

2. **用对话体写**：问题要像你真的坐在对面说出来的话，不是论文里的研究问题。可以用口语、可以八卦、可以吐槽。

3. **先铺垫再提问**：好问题 = 背景铺垫（1-2 句）+ 尖锐提问（1 句）。铺垫用来展示你做了功课，提问用来引出故事。

4. **一个问题一个方向**：不要在一个问题里塞 3 个子问题。如果你发现问题里有"另外""顺便""还有"，拆开它。

5. **制造情绪**：好问题让嘉宾想讲故事，不是想念 PPT。用对比、悬念、假设、挑衅来制造"我一定要回答这个"的冲动。

### 问题类型工具箱

| 类型 | 用途 | 示例 |
|------|------|------|
| 引用反弹 | 用嘉宾自己的话开头 | "你说过'一开始稳定生活就会想死'——这种想法是什么时候有的？" |
| 从A到B | 用跨度制造好奇 | "从帮《十三邀》拍林志玲，到白天在大学做AI、晚上煮奶瓶看Java——中间发生了什么？" |
| 反直觉 | 挑战常识 | "你以前收到Apple Watch都会退货，觉得科技是'人文的大敌'。当时怎么想的？" |
| 具体场景 | 让嘉宾回到现场 | "来深圳第一天就被问'你凭什么觉得你可以创造价值'——当时你站在哪？脑子里第一个念头是什么？" |
| 假设/选择 | 制造趣味 | "如果人生是一杯咖啡，你觉得瑞银时期的你是哪种？现在的你又是哪种？" |
| 快速追问 | 挖深一个点 | "等一下——你说一年采访两百多个人，有没有一个时刻让你意识到'原来人类可以这么奇怪'？" |

### 好问题 vs 坏问题

**破冰部分：**

| 坏（面试感） | 好（朋友感） |
|---|---|
| 请先介绍一下自己 | 听说你以前帮《十三邀》拍过林志玲？作为阅人无数的制片人，当时第一眼见她什么感觉？ |
| 您为什么选择现在这份工作？ | 你在 LinkedIn 上说只回有趣的 DM——能不能透露一下，什么样的开场白会让你'秒回'？ |
| 请谈谈您的职业背景 | 用十个不同的身份标签来描述自己 |

**主题讨论部分：**

| 坏（空泛） | 好（具体） |
|---|---|
| 请分享一下您的创业经历 | 你一个人跑到深圳，从零开始创立Firefly——帮想离开打工模式的人寻找"第三条路"。到深圳第一天就被问"你凭什么"，怎么挺过来的？ |
| 您对职场晋升有什么建议？ | 你说过跟老板要建partner关系——但你自己一开始也觉得老板"跟神一样，连开玩笑都不敢"，什么时候开始转变的？ |
| 您如何看待AI对行业的影响？ | 你第一次体验LLM的时候用了一个词："惊悚"。那个"身份危机"具体是什么？是写作者的？创作者的？还是一个人的？ |
| 创业中遇到过什么困难？ | 当你决定离开年薪诱人的投行岗位时，身边最"现实"的反对声音是什么？你是如何说服自己不去计算那些"沉没成本"的？ |
| 请问您是如何平衡工作和生活的？ | 30+ 宝妈，白天在大学做AI相关工作，晚上煮奶瓶看Java——哪个时间段是属于你自己的？ |

## 开场白写作

### 规则

1. 第一句固定：`子鱼：Hallo 大家好！欢迎回到破壁圆桌。我是子鱼。`
2. 第二句固定：`Lumi：我是 Lumi。` 后接一句 hook
3. Hook 要用嘉宾最抓人的一个具体细节，不是头衔
4. 子鱼和 Lumi 交替说话，谁说哪句根据内容自然分配
5. 总共不超过 5 行
6. 最后一句邀请嘉宾打招呼并自我介绍

### 示例对比

**坏（太正式）：**
> 子鱼：今天我们非常荣幸地邀请到了职场领域的资深专家Mike来跟我们分享他宝贵的经验和见解。

**好（有细节、有性格）：**
> Lumi：今天请到的嘉宾，对在海外打拼、或是计划去海外发展的华人朋友们来说，是一位含金量极高的职业导师。
>
> 子鱼：没错。他曾是亚马逊 L8 级别的总监，管理过超过 1000 人的全球团队。现在他投身创业，致力于帮助海外华人打破职场社交和晋升中的那层隐形天花板。

## 禁用词汇和表达

### 问题中禁用

- 请您谈谈... / 能否分享一下... / 请问您是如何...
- 您对...有什么看法 / 见解 / 感想？
- ...的意义是什么？
- 您是如何看待...的？
- 在您的职业生涯中...
- 作为一个...，您认为...

### 描述中禁用

- 深入探讨 / 深度解析 / 全面剖析
- 探索...的魅力 / 领略...的风采
- 分享了宝贵的经验和见解
- 碰撞出思想的火花
- 干货满满 / 收获颇丰
- 赋能 / 破圈 / 硬核
- 在...的道路上 / 在...的浪潮中
- 不仅...更... / 既...又...（排比堆砌）

### 替代策略

用具体事实代替修饰性形容词：

| 坏 | 好 |
|---|---|
| 拥有丰富的管理经验 | 管过1000+人的全球团队 |
| 取得了卓越的成就 | 从L6一路杀到L8总监 |
| 对行业有深刻的洞察 | 他认为华人的瓶颈不在语言，在于对权力规则的误读 |
| 创办了一家成功的公司 | 创立Firefly——帮想离开打工模式的人和传统行业老板配对 |

## 双语处理

- 主体语言为中文
- 英文人名、品牌名、专业术语保留原文
- 问题中可以自然中英混排（"你说你的approach是..."）
- 嘉宾的英文原话尽量保留英文，用引号标注
