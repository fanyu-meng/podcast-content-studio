# 采访大纲写作风格指南

## 核心原则

**真诚 > 八卦，了解人 > 秀调研，好奇 > 严肃**

采访大纲的目标是帮你真正了解一个人，让对话自然展开，让嘉宾讲出故事——不是列一个 FAQ，也不是把从网上扒来的细节做成问答测验。

---

## 嘉宾资料写作

### 规则

1. **核心标签**用 `/` 分隔，3-5 个关键词，突出身份反差或跨界
2. **嘉宾亮点**用 bullet points，每条带 bold 子标签
3. 每条亮点必须包含**具体数字或细节**，不说"丰富经验"
4. 亮点描述后要有 "so what" —— 为什么这对采访有价值
5. 如果能找到嘉宾的原话/金句，放在资料部分最前面

### 示例对比

**坏（简历体）：**
> **核心标签：** 资深管理者 / 创业者 / 职场导师
>
> * **管理经验：** 拥有多年大厂管理经验，带领过大型团队
> * **创业经历：** 创办了一家职场教育公司

**好（故事体）：**
> **核心标签：** 前亚马逊 L8 总监 / 硅谷高管教练 / 职场"非职权影响力"专家
>
> * **硬核背景：** 密歇根大学商学院/数学/经济多面手，从顶级 IC 一路杀到亚马逊 L8 总监，管理过 1000+ 人的全球团队。
> * **痛点洞察：** 深谙海外华人职场"天花板"的真相。他认为华人职场的瓶颈往往不在于语言，而在于**对权力游戏规则的误读**以及**沟通肌肉记忆的缺失**。

## 采访问题设计

### 五条铁律

1. **真诚好奇优先**：最好的问题往往很简单——"你的偶像是谁？""每天练多长时间？""为什么选德国？"。这些问题本身不需要调研细节，但嘉宾的回答只属于他自己。大约 60% 的问题应该是这类真诚、开放的问题。

2. **调研用于理解，不用于炫耀**：调研是帮你心里有数，不是每个问题都要把查到的东西塞进去。引用具体细节时（约 40% 的问题），优先用嘉宾自己说过的话，避免引用不可信的媒体转述。如果不确定某个细节是否属实，就别放进问题——直接问嘉宾本人。

3. **用对话体写**：问题要像你真的坐在对面说出来的话。简单问题就简单问，别加不必要的铺垫。复杂问题可以先铺垫（1-2 句）再提问（1 句）。

4. **一个问题一个方向**：不要在一个问题里塞 3 个子问题。如果你发现问题里有"另外""顺便""还有"，拆开它。

5. **问内心，不只问事实**：好问题触及感受、动机、犹豫——"突然不在身边是什么感觉？""有没有后悔过？"。纯事实核查式的问题（"你月薪多少？""那天第一个电话打给了谁？"）容易变成八卦，慎用。

### 问题类型工具箱

| 类型 | 用途 | 示例 |
|------|------|------|
| 引用反弹 | 用嘉宾自己的话开头 | "你说过'一开始稳定生活就会想死'——这种想法是什么时候有的？" |
| 从A到B | 用跨度制造好奇 | "从帮《十三邀》拍林志玲，到白天在大学做AI、晚上煮奶瓶看Java——中间发生了什么？" |
| 反直觉 | 挑战常识 | "你以前收到Apple Watch都会退货，觉得科技是'人文的大敌'。当时怎么想的？" |
| 具体场景 | 让嘉宾回到现场 | "来深圳第一天就被问'你凭什么觉得你可以创造价值'——当时你站在哪？脑子里第一个念头是什么？" |
| 假设/选择 | 制造趣味 | "如果人生是一杯咖啡，你觉得瑞银时期的你是哪种？现在的你又是哪种？" |
| 快速追问 | 挖深一个点 | "等一下——你说一年采访两百多个人，有没有一个时刻让你意识到'原来人类可以这么奇怪'？" |

### 好问题 vs 坏问题

**破冰部分：**

| 坏（面试感） | 好（朋友感） |
|---|---|
| 请先介绍一下自己 | 简单介绍一下在德国的现状：学校生活，德语环境，演出情况 |
| 您为什么选择现在这份工作？ | 你的吉他偶像是谁？ |
| 请谈谈您的职业背景 | 用十个不同的身份标签来描述自己 |

**主题讨论部分：**

| 坏（空泛 or 八卦） | 好（真诚 or 具体） |
|---|---|
| 请分享一下您的创业经历 | 你一个人跑到深圳，第一天就被问"你凭什么"，怎么挺过来的？ |
| 听说你月薪100多花了1100给儿子拉网线 | 退学那个决定是你自己提出来的，还是爸爸的主意？ |
| 拿到冠军那天晚上第一个电话打给谁？ | 欧洲吉他奖冠军，这个奖项对你意味着什么？ |
| 听说你逮着谁都问要不要听弹琴 | 什么时候开始独立创作的？你的创作习惯是怎样的？ |
| 请问您是如何平衡工作和生活的？ | 到了德国之后，第一次长期离开父母，突然不在身边是什么感觉？ |

## 八卦 vs 真诚：调研的正确用法

调研是为了让你心里有数，不是为了让问题更花哨。

| 八卦式问题（避免） | 真诚式问题（推荐） | 区别 |
|---|---|---|
| 听说你一怒之下从1楼弹到4楼？ | 走红那天，你当时的心态是什么？ | 前者转述媒体叙事，后者问内心感受 |
| 你爸月薪100多花了1100给你拉网线 | 你和你爸会有后悔退学这个决定的时候吗？ | 前者八卦家庭财务，后者问真实情感 |
| 你以前逮着谁都问要不要听你弹琴，连收水电费的阿姨都不放过 | 第一次接触吉他是什么时候？为什么选择指弹？ | 前者是小报式轶事，后者是了解人的起点 |
| 拿到冠军那天晚上第一个电话打给了谁？ | 这个奖项对你意味着什么？ | 前者追求戏剧性，后者问意义 |

**原则：**
- 如果一个细节来自媒体转述而非嘉宾本人，不要当作事实放进问题
- 如果一个问题更像是在满足听众的窥探欲而非帮助理解嘉宾，换掉
- 简单、开放的问题往往比精心设计的"钩子"更容易引出好故事

## 开场白写作

### 规则

1. 第一句固定：`子鱼：Hallo 大家好！欢迎回到破壁圆桌。我是子鱼。`
2. 第二句固定：`Lumi：我是 Lumi。` 后接一句 hook
3. Hook 要用嘉宾最抓人的一个具体细节，不是头衔
4. 子鱼和 Lumi 交替说话，谁说哪句根据内容自然分配
5. 总共不超过 5 行
6. 最后一句邀请嘉宾打招呼并自我介绍

### 示例对比

**坏（太正式）：**
> 子鱼：今天我们非常荣幸地邀请到了职场领域的资深专家Mike来跟我们分享他宝贵的经验和见解。

**好（有细节、有性格）：**
> Lumi：今天请到的嘉宾，对在海外打拼、或是计划去海外发展的华人朋友们来说，是一位含金量极高的职业导师。
>
> 子鱼：没错。他曾是亚马逊 L8 级别的总监，管理过超过 1000 人的全球团队。现在他投身创业，致力于帮助海外华人打破职场社交和晋升中的那层隐形天花板。

## 禁用词汇和表达

### 问题中禁用

- 请您谈谈... / 能否分享一下... / 请问您是如何...
- 您对...有什么看法 / 见解 / 感想？
- ...的意义是什么？
- 您是如何看待...的？
- 在您的职业生涯中...
- 作为一个...，您认为...

### 描述中禁用

- 深入探讨 / 深度解析 / 全面剖析
- 探索...的魅力 / 领略...的风采
- 分享了宝贵的经验和见解
- 碰撞出思想的火花
- 干货满满 / 收获颇丰
- 赋能 / 破圈 / 硬核
- 在...的道路上 / 在...的浪潮中
- 不仅...更... / 既...又...（排比堆砌）

### 替代策略

用具体事实代替修饰性形容词：

| 坏 | 好 |
|---|---|
| 拥有丰富的管理经验 | 管过1000+人的全球团队 |
| 取得了卓越的成就 | 从L6一路杀到L8总监 |
| 对行业有深刻的洞察 | 他认为华人的瓶颈不在语言，在于对权力规则的误读 |
| 创办了一家成功的公司 | 创立Firefly——帮想离开打工模式的人和传统行业老板配对 |

## 双语处理

- 主体语言为中文
- 英文人名、品牌名、专业术语保留原文
- 问题中可以自然中英混排（"你说你的approach是..."）
- 嘉宾的英文原话尽量保留英文，用引号标注
