# Show Notes 示例参考

阅读以下示例，学习写作风格和格式。每个示例后有简要分析，说明其写作手法。

---

## 示例 1：EP15（脱口秀主题）

```
EP15：从每个月40场开放麦到喜剧荒漠🇩🇪：我们在慕尼黑讲脱口秀

单集封面：


『本期嘉宾』
林盐盐：前喜剧联盒国签约演员，在读人工智能硕士
东东：前慕尼黑椒糖喜剧主理人，现役作家

『主要内容』
非常开心和前喜剧联盒国的资深脱口秀演员林老师@林盐盐，以及@慕尼黑椒糖喜剧的前主理人东东老板@邵东今天也好运，展开了亲切会晤。会议在慕尼黑高端会所Fitzroy举办，气氛热烈祥和。会议期间，二位嘉宾对中英文脱口秀的发展和国内外脱口秀的行业动向都做了洞见十足的见解和分析，并对德国美食和德国的苦逼生活进行了辛辣讽刺。会议在慕尼黑罕见的晚霞中落下帷幕。

录制于慕尼黑最fancy的Cafe：FitzRoy Munich

fancy的景色也掩盖不住我寒酸的设备


『时间戳』
00:13 东东的身份转场：从厦门酒吧"精酿啤酒"主理人到移居德国

01:02 盐盐的双重身份：从"喜剧联盒国"演员到慕尼黑AI在读大学生

02:29 喜剧启蒙时刻：盐盐在爱丁堡边缘艺术节（Fringe Festival）首次感受单口喜剧的魔力

07:58 栋笃笑冲击：黄子华四面台演出为什么让人创作观被重置？

10:58 厦门喜剧往事：在自家酒吧偶遇庞博、杨笠，早期英文脱口秀据点就此成形

14:43 题材差异：海外英文开放麦常聊"移民身份"，中文脱口秀更追求作品完整性

19:00 创立"椒糖喜剧"：东东把线本文学平台运营经验变成"经纪人"思维，慕尼黑中文开放麦为何场场爆满

22:17 场地执念：为什么宁可在教堂演出，也要拒绝有"炒菜声"的场地？

26:43 台上崩溃谜题：为什么生活里的幽默大师，一上台反而笑不出来？

30:56 成长曲线：盐盐如何从"只能撑三分钟"到"每分钟都有笑点"

31:41 "卷王"经历：国内演出市场顶峰期，一个月讲40多场开放麦？！

34:06 现实拐点：讲脱口秀，真的赚到钱了

38:15 价值拉扯：搞笑vs上价值，取悦观众还是取悦自己？

43:56 风格坐标：从赵本山到黄子华，从Iliza Shlesinger到Fluffy

48:37 情绪共鸣：喜欢杨笠和小鹿，为什么会看脱口秀看哭？

50:16 东东泪点：哪个椒糖喜剧演员真的让他破防？

53:10 终极追问：中国脱口秀何去何从？我们定一下子
```

**手法分析：**
- 标题用了具体数字"40场"作为 hook，对比"喜剧荒漠"形成反差
- 主要内容用"新闻联播"播报体戏仿，"亲切会晤"、"气氛热烈祥和"制造反差幽默
- 夹带私货："fancy的景色也掩盖不住我寒酸的设备" —— 自嘲
- 时间戳统一为 `MM:SS 标题关键词：具体描述`，并在条目间保留空行，便于直接复制
- "一个月能讲40多场开放麦？！"用标点传达情绪

## 示例 2：嘉宾介绍（AI创业主题）

```
『本期嘉宾』
Cathy Liu，连续创业者，也曾是"职场打工人"，做过公司的VP、COO。她从程序员开始一路走来，从清华走到硅谷。现为solopreneur.global的创始人，为一人公司或小团队服务的AI应用工厂和创业平台。帮助创业者缩短从想法到PMF(Product-Market Fit)验证的时间，加速MVP落地以及市场探索，快速得出go/no-go的结论。
Cathy做过从0到1的创业，也曾作为公司高管带领业务从0做到20亿人民币，她还曾以COO身份推动A轮公司扩张，实现规模化增长。
她是硅谷创业者，也是两个孩子的妈妈。
如果你对AI创业感兴趣，无论是想找点子、加速实现、验证市场或是寻找同路人，欢迎访问并注册： https://solopreneur.global
```

**手法分析：**
- 嘉宾介绍是叙事体而非简历体，读起来像故事不像 bio
- "从程序员开始一路走来，从清华走到硅谷"——用路径串联身份
- 最后一句"她是硅谷创业者，也是两个孩子的妈妈"——人味收尾
- 附带链接和 CTA，方便听众行动

## 示例 3：EP13（脱口秀主题）

```
Title：
EP13：她用中英德三语讲脱口秀——逗笑德国人有这么难？

单集封面：

『本期嘉宾』
李刃，全职脱口秀演员，中英德三语表演者，🇩🇪德国汉堡laugh matter英文脱口秀俱乐部主理人，北京孩子。
多次获得欧洲脱口秀比赛冠军🏆：
荷兰「Comedy Talent Award」冠军；
德国Comedy Grand Slam冠军（2024、2025连续两年）；
卢森堡Comedy Battle Royale冠军
拥有个人英文专场「40% Friends」，多次售罄，德国持续巡演中。

个人网站：https://www.abilicomedy.com/
英文专场：https://www.abilicomedy.com/special
Instagram：https://www.instagram.com/abi.li.comedy/

『主要内容』：
李刃是全职脱口秀演员，也是一家俱乐部的老板。从创业失败到第一次登上脱口秀舞台，从几个月没人笑到获得欧洲各项脱口秀比赛冠军，从"Pround failure"到舞台上的闪闪发光，李刃都经历了什么？脱口秀对她意味着什么？用不同语言讲脱口秀有什么不同？如何进行创作？

这期可能是录制以来我们主持人说话最多的一次，也是笑声最多的一次。李刃是个非常有人格魅力的人，更是有趣的人，社交悍匪，随随便便就能输出风趣的幽默。如果我们的笑声过大过于刺耳，李刃全责。
```

**手法分析：**
- 标题用破折号制造悬念："逗笑德国人有这么难？"
- 嘉宾介绍：先一句话概括身份，再列成就（用emoji点缀），附链接
- 主要内容第一段用"从...到..."排比制造叙事弧线，以提问收尾引发好奇
- 第二段切换到主持人视角的主观评价，充满人情味
- "社交悍匪"、"李刃全责"——口语化、有个性的表达
